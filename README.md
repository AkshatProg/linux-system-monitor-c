# Linux System Monitor in C

A simple Linux System Monitor built in C using the Linux proc filesystem.

## Features

- CPU Information
- Memory Information
- Memory Usage Calculation
- System Uptime

## Linux Interfaces Used

- /proc/cpuinfo
- /proc/meminfo
- /proc/uptime

## Concepts Practiced

- Functions
- File Handling
- fopen()
- fclose()
- fgets()
- fscanf()
- sscanf()
- String Parsing
- Linux Proc Filesystem
